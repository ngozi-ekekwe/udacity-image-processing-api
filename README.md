## Overview
Simple Node API for resizing images using [!Sharp](https://www.npmjs.com/package/sharp)

## Requirements

Node LTS VERSION


## How to start server

```
 - npm install 
 - npm start

 cd client
 - npm install 
 - npm start
```


## Testing

```
 - npm run test
```
